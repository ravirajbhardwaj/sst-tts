## Steps

1. **Speak:**  
   Talk into your microphone.

2. **Transcribe:**  
   Speech is converted to text.

3. **Reply:**  
   Text is sent to Gemini API for a response.

4. **Playback:**  
   The reply is spoken aloud.

## Tech Used

- `SpeechRecognition` (for SST)
- `SpeechSynthesisUtterance` (for TTS)
- Gemini API (for replies)
